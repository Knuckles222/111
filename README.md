[플레이 방법]
1. 게임 설명서를 읽고 게임 안의 기능(퍼즐요소)을 이해해주세요.
2. 게임 안의 스토리와 컷 씬을 보고 세계관을 이해해주세요.
3. 픽셀 그림판에서 자신이 원하는 디자인으로 백신을 이쁘게 디자인 해주세요.
4. 게임 조작법을 이해하고 머리를 써서 퍼즐을 열심히 풀어주세요.

[유의사항]
- 퍼즐은 무조건 제작자가 정해놓은 길로 가야지만 클리어가 할 수 있게 만들었습니다.
- 퍼즐은 무조건 풀리게 되어있으니 게임 기믹들을 이해하고 재밌게 게임을 즐겨주세요.
- 저장/불러오기 기능이 구현되어 있습니다. 게임 진행 정도를 저장해야하는 상황이 오면 esc메뉴에서 꼭 저장을 눌러주세요.
